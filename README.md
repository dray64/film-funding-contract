# 🎬 ChainCinema — Decentralized Film Funding on Stellar

## 📌 Project Description

ChainCinema is a fully permissionless decentralized application (DApp) built on the Stellar network using Soroban smart contracts. It enables filmmakers to raise funds directly from a global community without relying on studios, producers, or centralized platforms.

By leveraging blockchain technology, ChainCinema ensures transparency, trustless execution, and open participation — allowing anyone to create, fund, and participate in film projects.

---

## 🚀 What It Does

- 🎥 Allows creators to launch film funding campaigns  
- 💰 Enables anyone to fund projects directly using their wallet  
- 🔐 Uses smart contracts to securely hold and manage funds  
- ⏳ Enforces deadline-based funding rules automatically  
- 🔁 Provides refunds if funding goals are not met  

---

## ✨ Key Features

### 🔓 Fully Permissionless
- No admin or central authority  
- Anyone can create a project  
- Anyone can fund any project  

---

### 💸 Trustless Escrow System
- Funds are locked inside the smart contract  
- Released only if funding conditions are met  
- Eliminates need for trust between users  

---

### ⏳ Deadline-Driven Logic
- Each project has a funding deadline  
- Smart contract enforces:
  - Successful funding → creator can claim funds  
  - Failed funding → contributors can refund  

---

### 📊 Transparent & On-Chain
- All project data stored on-chain  
- Contributions are publicly verifiable  
- Fully auditable system  

---

### 🧩 Modular & Extensible
Designed to support future upgrades:
- NFT-based ownership for investors  
- DAO governance for decision-making  
- Milestone-based funding release  
- Reputation systems for creators
